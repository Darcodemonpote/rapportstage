# Outils et Environnements

## Outils utilisés

- **Talend** : développement de jobs ETL
- **Hue** : requêtage SQL sur les données Hadoop
- **Jenkins** : CI/CD
- **Yarn**, **Administrator Center / Real-Time**
- **Jira** : gestion de tickets
- **Git** : versioning (commit dès le 1er jour)
